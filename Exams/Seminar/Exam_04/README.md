# Контролно 4 - Структури от Данни и Алгоритми

## Задача 1 (60 точки)

Даден е масив с размер **N** от цели числа и цяло число **K**.  
Да се намери **дължината на най-дългия подмасив**, в който **всеки елемент се среща най-много K пъти**.

### Вход:

- На първия ред се въвеждат две цели числа **N** и **K**
- На втория ред се въвеждат елементите на масива

### Изход:

- Изведете едно число - дължината на най-дългия търсен подмасив

### Пример:

**Вход:**

```
8 2
1 3 4 1 3 2 1 1
```

**Изход:**

```
6
```

**Обяснение:**
И двата подмасива
(от индекс 0 до 5 → `1 3 4 1 3 2`)
и
(от индекс 1 до 6 → `3 4 1 3 2 1`)
са валидни и имат максимална дължина 6.

## Задача 2 (40 точки)

Играете настолната игра **„Котан“**.
Картата е матрица **N × N**, започвате от `(0, 0)` и трябва да стигнете до `(N-1, N-1)`.

Всяко поле `(i, j)` има вероятност **P[i][j]** капан да се задейства при стъпване върху него.
Ако капан се задейства – губите моментално.

### Движение:

- Можете да се движите само **надясно** или **надолу**.
- Задействането на капан е независимо за всяко поле.
- **P[0][0] = 0**, **P[N-1][N-1] = 0**.

Намерете **минималната вероятност за загуба**, ако изберете възможно най-безопасния път.

### Вход:

- На първия ред: цяло число **N**
- Следват N реда с по N числа: **P[i][j]** - числа с плаваща запетая в интервала [0, 1].

### Изход:

- Едно число - минималната вероятност да загубите.

### Пример:

**Вход:**

```
3
0 0.25 0.5
0.4 0.5 0.5
0.1 0 0
```

**Изход:**

```
0.46
```

**Обяснение:**
Най-безопасният път е:
`(0,0) -> (1,0) -> (2,0) -> (2,1) -> (2,2)`

Има два капана:

- 40% вероятност → шанс за оцеляване 0.6
- 10% вероятност → шанс за оцеляване 0.9

Общ шанс за оцеляване:
`0.6 * 0.9 = 0.54`

Следователно шансът за загуба е:
`1 - 0.54 = 0.46`
