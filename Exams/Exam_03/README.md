## Задача 2 (50т)

Разполагаме с двоично дърво за търсене (BST), което съдържа цели числа. Дървото първоначално е празно. В него последователно се добавят **N** на брой различни цели числа (без да се премахват елементи).

Напишете програма, която намира кое е **най-малкото число** в дървото, което е **строго по-голямо** от дадено число **X**. Ако в дървото няма число, което да е строго по-голямо от **X**, програмата трябва да изведе **NO**.

### Входни данни:

От стандартния вход се въвежда:
- На **първия ред** — едно цяло число **N** - брой елементи в дървото.
- На **втория ред** — **N** различни цели числа **Ai** (разделени с интервал) в реда, в който се добавят към дървото.
- На **третия ред** — едно цяло число **X**.

### Изходни данни:

Изведете на стандартния изход точно един ред:
- Най-малкото число в дървото, което е строго по-голямо от **X**.
- Или низът **NO**, ако такова число не съществува.

### Ограничения:

- `1 < N < 200000`
- `-10^9 ≤ Ai ≤ 10^9`
- `-10^9 ≤ X ≤ 10^9`

### Примери:

**Пример 1:**

```
Вход:
5
5 1 9 3 7

Изход:
7
```

Дърво:
```
      (5)
     /   \
   (1)   (9)
     \   /
     (3)(7)  <-- Отговорът
```

---

**Пример 2:**

```
Вход:
4
10 4 8 1

Изход:
10
```

Дърво:
```
     (10)  <-- Отговорът
     /
   (4)
   / \
 (1) (8)
```

---

**Пример 3:**

```
Вход:
4
10 4 8 1
10

Изход:
NO — няма строго по-голямо число от 10 в дървото
```

Дърво:
```
     (10)
     /
   (4)
   / \
 (1) (8)
```
